แผนการเรียนรู้ (Official Path)

หลักสูตรตามมาตรฐาน Hiwonder เพื่อการใช้งาน MentorPi M1 อย่างเต็มประสิทธิภาพ

วันที่ 1: พื้นฐานวิศวกรรมและวิสัยทัศน์ AI (Full Day)

บทที่ 1: การเตรียมความพร้อมและควบคุมพื้นฐาน →

การเชื่อมต่อ Wi-Fi (AP/LAN), การรีโมท (VNC/SSH), การใช้งาน Docker และการควบคุมล้อ Mecanum

บทที่ 2: ระบบเซนเซอร์และ Lidar เบื้องต้น →

การทำงานของ Lidar (STL-19P), Workshop หุ่นยนต์เฝ้ายาม (Guarding) และการหลบหลีกสิ่งกีดขวาง

บทที่ 3: Computer Vision & MediaPipe →

การประมวลผลภาพด้วย OpenCV, การเขียนโปรแกรมวิ่งตามเส้น และการตรวจจับท่าทางมือ/ใบหน้า

บทที่ 4: Machine Learning (YOLO) →

การใช้งาน YOLOv5/v11 บนหุ่นยนต์เพื่อการตรวจจับวัตถุและป้ายจราจรในระดับ Real-time

วันที่ 2: ระบบนำทางอัตโนมัติ (Half Day)

บทที่ 5: 2D SLAM & Localization →

การสแกนพื้นที่เพื่อสร้างแผนที่ด้วย slam_toolbox และการระบุตำแหน่งหุ่นยนต์ด้วย AMCL

บทที่ 6: Autonomous Navigation (Nav2) →

การวางแผนเส้นทาง (Path Planning) ด้วย DWA Algorithm และภารกิจการนำทางหลายจุด

บทที่ 7: Final Project →

ภารกิจรวม: การขับขี่อัตโนมัติในสนามจำลอง และตรวจจับสัญญาณจราจร
