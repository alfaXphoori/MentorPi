แผนการเรียนรู้ (Official Path)

หลักสูตรตามมาตรฐาน Hiwonder เพื่อการใช้งาน MentorPi M1 อย่างเต็มประสิทธิภาพ

วันที่ 1: สถาปัตยกรรมและการควบคุมพื้นฐาน

บทที่ 1: การเตรียมความพร้อม (Environment) →

การเชื่อมต่อ Wi-Fi (AP/LAN Mode), การรีโมทผ่าน VNC/SSH และการจัดการ Container ผ่าน Docker Integration

บทที่ 2: กลไกการเคลื่อนที่และเซนเซอร์

การควบคุม Mecanum Chassis, การปรับแต่ง Servo Deviation และการทำ IMU/Velocity Calibration

บทที่ 3: ระบบ Lidar เบื้องต้น

การทำงานของ Lidar (STL-19P), Workshop หุ่นยนต์เฝ้ายาม (Guarding) และการหลบสิ่งกีดขวาง (Avoidance)

วันที่ 2: วิสัยทัศน์คอมพิวเตอร์และปฏิสัมพันธ์ AI

บทที่ 4: OpenCV & Vision Logic

การประมวลผลภาพ (Color Recognition), การตรวจจับ QR Code และการเขียนโปรแกรมวิ่งตามเส้น (Line Following)

บทที่ 5: MediaPipe Interaction

การตรวจจับท่าทางมือ (Fingertip Recognition), การติดตามใบหน้า และระบบควบคุมหุ่นยนต์ด้วยท่าทางร่างกาย

บทที่ 6: Machine Learning Deployment

การใช้งาน YOLOv5/v11 บนหุ่นยนต์: ตั้งแต่การเตรียม Dataset ไปจนถึงการตรวจจับวัตถุในระดับ Real-time

วันที่ 3: ระบบนำทางและการขับเคลื่อนอัตโนมัติ

บทที่ 7: 2D SLAM & Localization

การสร้างแผนที่ด้วย slam_toolbox, การตั้งค่า URDF Model และการระบุตำแหน่งด้วย AMCL

บทที่ 8: Autonomous Navigation (Nav2)

การวางแผนเส้นทาง (Path Planning) ด้วย DWA Algorithm และภารกิจ Multi-point Navigation

บทที่ 9: Project: Autonomous Driving

ภารกิจรวม: การขับขี่อัตโนมัติในสนามจำลอง ตรวจจับสัญญาณไฟจราจร และการจอดรถอัตโนมัติ
