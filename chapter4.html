บทที่ 4: Machine Learning (YOLO)

4.1 ทำความรู้จัก YOLO (You Only Look Once)

YOLO คือโครงข่ายประสาทเทียม (Neural Network) ที่เก่งด้านการตรวจจับวัตถุ (Object Detection) ในระดับ Real-time ด้วยประสิทธิภาพของบอร์ด Raspberry Pi 5 ทำให้ MentorPi M1 สามารถรันโมเดล YOLOv5 หรือ YOLOv11 ได้โดยตรง

  • ข้อดี: ตรวจจับวัตถุได้หลายชนิดพร้อมกัน, บอกตำแหน่ง (Bounding Box) ได้แม่นยำ, ประมวลผลเร็ว

4.2 Workshop: Traffic Sign Detection

ในบทนี้ เราจะเปิดใช้งานโมเดล AI ที่ถูกเทรน (Train) มาแล้วเพื่อตรวจจับ "ป้ายจราจร" และ "สัญญาณไฟ" ซึ่งเป็นหัวใจสำคัญของการทำรถยนต์ไร้คนขับ (Autonomous Driving)

  1. เข้าสู่ระบบ Docker ผ่าน VNC Desktop ./enter_docker.sh
  2. พิมพ์คำสั่งเพื่อเรียกใช้โมเดล YOLO:
    ros2 launch yolo_app yolo_detection.launch.py
  3. นำแผ่นป้ายจราจรจำลอง (เช่น ป้ายหยุด, เลี้ยวซ้าย, เลี้ยวขวา) มาแสดงหน้ากล้องของหุ่นยนต์
  4. บนหน้าจอ VNC จะปรากฏกรอบสี่เหลี่ยมคลุมป้ายจราจร พร้อมแสดงชื่อคลาส (Class Name) และความมั่นใจ (Confidence Score)

💡 Tip: ROS2 จะส่งข้อมูล "สิ่งที่ตรวจพบ" ไปยัง Topic (เช่น /yolo/detections) นักเรียนสามารถเขียนโค้ด Python มารับค่านี้ เพื่อสั่งให้ล้อเบรกเมื่อเจอป้าย "STOP" ได้ในภายหลัง
